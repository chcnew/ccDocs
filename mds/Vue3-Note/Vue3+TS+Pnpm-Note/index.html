    Basic¶

    一、项目创建命令(pnpm)¶

    项目名：basic

    模板：vue-ts (文档：https://cn.vite.dev/guide/)

    在线IDE：https://stackblitz.com/edit/vitejs-vite-u5ajgo2v?file=index.html&terminal=dev

    npm install -g pnpm
pnpm create vite basic --template vue-ts
# 进入项目进行初始化
cd basic
pnpm install

    二、学习TypeScript¶

    • 开发环境测试ts脚本

pnpm add -D typescript ts-node

# 如果你想同时安装 Node.js 类型定义文件（推荐，用于获得更好的类型提示）
pnpm add -D @types/node

    • WebStorm 中配置运行

    • 右上角添加配置，选择node.js

    • 输入名称：runTS；选择scrip文件，即可

    1 数据类型¶

    // 自定义类型
type OderId = number

// 联合类型
type OderId = number | string | bigint

// 定义引用类型 object
type Gender = "man" | "woman";  // tS支持定义枚举类型

type Person = {
    name: string;
    age: number;
    gender: Gender;
    others: {
        hobby: string[];
    };
};

type PigColor = "white" | "black"
type Pig = {
    name: string;
    age: number;
    color: PigColor
}

// 定义交叉类型（类型并集） -- Animal含有Person+Pig全部字段
type Animal = Person & Pig

    2 泛型定义¶

    泛型可以实现：在实例化类型的时候，可以设置对应的类型

    2.1 不限制定义字段泛型¶

    不限制字段名必须有：hobby

    type Person<T> = {
    name: string;
    age: int;
    gender: Gender;
    others: T;
};

const person = Person<{ hobby: string[] }>{
    name: "chc",
    age: 29,
    gender: "man",
    others: {
        hobby: ["唱歌", "跳舞"]
    }
}

    2.2 限制定义字段泛型¶

    限制字段名必须有：hobby

    // 定义引用类型 object
type Gender = "man" | "woman";  // tS支持定义枚举类型

type Person<T extends { hobby: string[] }> = {
    name: string;
    age: number;
    gender: Gender;
    others: T;
};

// hobby字段必须有
const person: Person<{ hobby: string[]; salary: number }> = {
    name: "chc",
    age: 29,
    gender: "man",
    others: {
        hobby: ["唱歌", "跳舞"],
        salary: 20000
    }
}

console.log(person);

    2.3 定义默认字段泛型¶

    // 定义引用类型 object
type Gender = "man" | "woman";  // tS支持定义枚举类型

// 默认字段泛型
type Person<T = { hobby: string[] }> = {
    name: string;
    age: number;
    gender: Gender;
    others: T;
};

// hobby字段必须有
const person: Person = {
    name: "chc",
    age: 29,
    gender: "man",
    others: {
        hobby: ["唱歌", "跳舞"],
    }
}

console.log(person);

    2.4 extends关键字使用场景¶

    关键字 extends 主要使用场景：

    1. 继承
    2. 定义泛型
    3. 条件类型
    2.4.1 继承¶
    type Gender = 'male' | 'female';

class Person {
    name: string;
    age: number;
    gender: Gender;
}

class Teacher extends Person {
    college: string;
}

class Student extends Person {
    grade: number;
}

const student: Student = {
    name: "小明",
    age: 18,
    gender: "male",
    grade: 8
};

console.log(student);
    2.4.2 泛型约束¶
    function Say<T extends { id: number; name: string }>(info: T) {
    console.log(info);
}

// 必须包含泛型约束字段：id、name
Say({id: 100, name: "Hello World!", hobby: ["sing song", "play game"]})
    2.4.3 条件类型¶

    三目运算符确定条件类型

    type IsString<T> = T extends string ? true : false;

const isString: IsString<string> = true;  // 只能是：string+true 或 其他类型+false 否则报错

console.log(isString);

    3 内置类型工具(Pick/Omit/Readonly...)¶

    type Person = {
    name: string;
    age: number;
    gender: string;
}

// 创建一个类型，只需要Person里面的某些字段或设置为只读：使用内置类型工具
type NamePerson = Pick<Person, 'name'>;  // 只取某些字段
type AgePerson = Omit<Person, 'name' | 'age'>;  // 排除某些字段
type ReadonlyPerson = Readonly<Person>;

const agePerson: AgePerson = {
    gender: 'male',
}

console.log(agePerson);

    4 类型保护¶

    判断类型作出相应的操作

    type Index = number | string;

function isNumber(x: unknown): x is number {
    return typeof x === "number"
}

let num = 0
const index: Index = "123"
// const index: Index = 123

// index 不是数字类型则自动转换
if (isNumber(index) === false) {
    console.log('index is not a number')
    num = Number(index)
} else {
    num = index
}

console.log('num', num)

    三、vue3使用TS定义类型约束¶

    <template>
  <h2>name: {{ name }}</h2>
  <h2>age: {{ age }}</h2>
  <h2 @click="isOpen = !isOpen">{{ isOpen ? '点击关闭' : '点击开启' }}</h2>
  <p>
    {{ personInfo?.age}}
  </p>
</template>

<script setup lang="ts">
import {defineProps, ref} from 'vue';

// 宏定义 - 父传子 - 类型约束
defineProps<{
  name: string;
  age: number
}>();

// 开关状态显示 - 类型约束
const isOpen = ref<boolean>(false);

// 数据信息 - 类型约束
const personInfo = ref<{
  name: string;
  age: number
}>();
</script>
